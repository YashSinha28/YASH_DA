# Data Cleaning & Visualization Project

## Project Overview
This project demonstrates an end-to-end data science workflow designed to clean, process, and analyze raw, messy data. Using Python, the project takes raw datasets containing structural flaws, transforms them into high-quality data, and extracts actionable business insights through visualization.

### Key Features Applied:
* **Data Preprocessing:** Handled missing values via median/placeholder imputation.
* **Anomaly Detection:** Identified and removed duplicate records and extreme entry outliers (e.g., typos in numerical values).
* **Feature Engineering:** Developed custom metrics (Total Sales) for better financial evaluation.
* **Data Storytelling:** Generated clear, presentation-ready visual reports using `Matplotlib` and `Seaborn`.

### Tools Used:
* Python 3
* Pandas & NumPy
* Matplotlib & Seaborn
* Jupyter Notebook
